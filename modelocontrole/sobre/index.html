Sobre o Laboratório de Simulação

Iniciativa da Engenharia de Controle e Automação (UFSC) para ensino e pesquisa em sistemas térmicos e energéticos, com foco em modelagem, controle e integração de dados.

Coordenação

Prof. Marcos Vinícius Americano

Universidade Federal de Santa Catarina • Programa INCT

Missão

Democratizar o acesso a simuladores web responsivos para disciplinas e projetos de pesquisa, conectando modelagem matemática, controle (PID/MPC) e dados reais.

Objetivos
  • Desenvolver simuladores acessíveis via navegador (desktop e mobile);
  • Explorar estratégias de controle e visualização de dados;
  • Disponibilizar perfis externos via CSV e reprodutibilidade dos experimentos.
Âmbitos
  • Sistemas Térmicos (coletores solares, caldeiras);
  • Processos Eletroquímicos (eletrolisadores);
  • Integração com energias renováveis e instrumentação.

Tecnologias

Stack
  • HTML5, CSS3 (Bootstrap 5 + CSS custom UFSC)
  • JavaScript (modelo numérico, PID/MPC)
  • Chart.js para gráficos
  • Perfil externo via CSV (interp. linear por horário)
Arquitetura
  • SPA leve por página de simulador
  • Execução em lote (tempo simulado) e loop contínuo (tempo real)
  • Componentes reusáveis de UI (modais, toasts) e estilos UFSC

Perguntas Frequentes

Sim. Os simuladores aceitam arquivos CSV com horários e perfis (ex.: irradiação, temperatura).

O PID é o padrão. Estamos avaliando extensões (ex.: MPC) conforme o projeto avança.
